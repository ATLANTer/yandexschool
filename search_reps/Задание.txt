GitHub.com предоставляет API для работы с репозиториями, пользователями, проектами и т.е.
Нужно написать скрипт, который выполняет следующие действия: 
1. по введенному слову ищет репозитории, которые содержат это слово в своем названии 
(документация https://docs.github.com/en/rest/reference/search#search-repositories), 
2. сортирует их по количеству звезд в порядке убывания, 
3. записывает полученный список в указанный json файл.

Параметры скрипта:
-v, --value - строка для поиска
-f, --file - путь к выходному json файлу; если файл уже существует, нужно показывать уведомление с предложением его перезаписать; 
если пользователь отказывается перезаписывать файл, то скрипт прекращает работу.

Пример запуска:
python search_reps.py -v tetris -f C:\files\reps.json

Используемые пакеты/модули:
os, os.path, requests, json, argparse
